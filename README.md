# Calculadora de Carbono Avançada

Projeto Python/Streamlit com:
- Temas: Light, Dark, UNIP
- Escopos: Pessoal, Empresarial, Híbrido
- Idiomas: Português, English, Bilingue (toggle)

### Como rodar
1. Instale dependências:
```
pip install -r requirements.txt
```
2. Abra VS Code, terminal e rode:
```
streamlit run app/main.py
```
